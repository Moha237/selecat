# 011.11 Compute the tangent line of a function at a given point

### Description: 
To find the tangent line of a function at a given point, you need to:
1. Compute the derivative f'(x)
2. Evaluate it at the point of interest: f'(a)
3. Use the point-slope formula for a line:
   - y - f(a) = f'(a)(x - a)

### Example: 
**Problem**:
Consider the function \( f(x) = x e^{x-1} \). Calculate the equation of the tangent line to the graph of the function \( f \) at the point with abscissa \( x = 1 \).  

**Solution**:
1. Compute the derivative:
[comment]: #Here we should tell the student that it has to use the product rule.
    f'(x) = (1 + x) e^{x-1}

2. Evalutate the derivative at x = 1:
   f'(1) = 2

3. Evaluate the function at x = 1:
    f(1) = 1 \times e{1-1} = 1

4. Use the point-slope formula:
   y - 1 = 2(x - 1)

5. Final answer:
6. y = 2x - 1
